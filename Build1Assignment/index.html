Project Name: WorldSurf

Project Description

This first project in WiU class requires us to set up a landing page. We can make our own based on the template given or modify the template to achieve our final product. A few other things that are required of each student is accumulating at least 25 karma points on navflow.com. We also need to style the page in our own way and make sure the HTML and CSS validate via the W3C validators. Once these items are complete, this should be pushed up to our Github account
